

## Project Name 

#### Pomodoro Timer

An Pomodoro Timer Application, built with Vite, React, JavaScript, and CSS.

## Installation and Setup Instructions
 

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Start Server:

`npm run dev`   

To Visit App:

`localhost:3000`
